# 📚 Quiz Português - Flutter

Um aplicativo mobile desenvolvido em Flutter com o objetivo de testar conhecimentos em Língua Portuguesa de forma interativa e dinâmica.

O quiz aborda conteúdos como **escolas literárias**, **gramática** e **funções da linguagem**, apresentando ao final o desempenho do usuário com base na quantidade de acertos.

---

## 🎯 Funcionalidades

- Tela inicial com acesso ao quiz  
- Questões de múltipla escolha  
- Feedback visual ao selecionar respostas (correto/incorreto)  
- Barra de progresso durante o quiz  
- Tela final com resultado e porcentagem de acertos  
- Navegação simples e intuitiva entre as telas  

---

## 🧠 Conteúdos abordados

- Escolas literárias (Arcadismo, Barroco, Romantismo, etc.)  
- Movimentos literários brasileiros  
- Interpretação de conceitos literários  
- Gramática (crase, funções da linguagem, etc.)

## ▶️ Como executar o projeto

Siga os passos abaixo para rodar o projeto localmente:

### 1. Clone o repositório
```bash
git clone https://github.com/mariaeduardaurbano/flutter_quiz.git
```

### 2. Acesse a pasta do projeto
```bash
cd flutter_quiz
```

### 3. Instale as dependências
```bash
flutter pub get
```

### 4. Execute o projeto
```bash
flutter run
```

> Certifique-se de ter o Flutter instalado e um emulador ou dispositivo conectado.
